# Yt-dlp-manager

Simple program to manage a download queue for [Yt-dlp](https://github.com/yt-dlp/yt-dlp/)

By default only has one concurrent download, as that is what i need, but can be increased by changing [this line](https://github.com/dani211e/Yt-dlp-manager/blob/4b9a22dbfa511239e3e758dba680b888fd1dff44/Yt-dlp-manager/Downloads/DownloadManager.cs#L7)
